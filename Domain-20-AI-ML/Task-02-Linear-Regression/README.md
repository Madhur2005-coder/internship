# Task 02 - Linear Regression

This task demonstrates how linear regression can predict student exam scores from study-related factors.

## Features Used

- Study hours per day
- Attendance percentage
- Sleep hours per night
- Class participation score
- Assignment completion percentage

## Run

```bash
python linear_regression.py
```

The script trains a linear regression model, predicts scores for a test split, and prints model coefficients plus sample predictions.
